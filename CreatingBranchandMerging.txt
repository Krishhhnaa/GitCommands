1. firstly create a new branch using --> git branch kunal

2. then come to this branch using --> git checkout kunal

3. then do whatever things you want to do here and afer that all the changes that have been made we want to merge the changes into main/master branch

4. to do that we will now switch to master branch using --> git checkout master

5. then we will fetch the changes using --> git fetch

6. then we will merge the branch using --> git merge kunal

7. after all this we will push the changes to repo using --> git push origin master
